# Aula TypeScript: Type Aliases, Tipagem de Objetos e Propriedades Opcionais

## Descrição
Este projeto contém exemplos práticos sobre como utilizar Type Aliases, objetos tipados e propriedades opcionais no TypeScript.  
O conteúdo faz parte do blog **guilhermemachadodev**, que documenta o aprendizado contínuo em desenvolvimento TypeScript.

## Conteúdo
1. **Type Aliases:** criação de apelidos de tipos para evitar repetição e melhorar a legibilidade.  
2. **Tipagem de Objetos:** definição de estruturas de objetos com tipos estáticos e seguros.  
3. **Propriedades e Parâmetros Opcionais:** uso do operador `?` para tornar campos e argumentos não obrigatórios.

## Como Executar
